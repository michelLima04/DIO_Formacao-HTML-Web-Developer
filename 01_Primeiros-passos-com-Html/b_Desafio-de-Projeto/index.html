1. Tags

Aprendi que as tags formam a estrutura básica de qualquer página web. Toda página começa com a tag <html>, possui um <head> para configurações (como o título da página) e um <body> onde colocamos o conteúdo visível.

Também vi como criar cabeçalhos usando tags de <h1> a <h6> e parágrafos usando a tag <p>. Além disso, aprendi a deixar comentários no código que não aparecem na tela usando <!-- comentário -->.

3. Textos

O HTML possui várias tags para formatação e semântica de textos. Algumas das que testei foram:

  • <strong>: Deixa o texto em negrito.
  • <i>: Deixa o texto em itálico.
  • <u>: Deixa o texto sublinhado.
  • <mark>: Cria um texto destacado (marca-texto).
A tag <blockquote> é excelente para fazer citações como esta!

4. Listas

Podemos organizar informações em listas. A Lista Não Ordenada (tag <ul>) usa marcadores simples:

  • Maçã
  • Banana

Já a Lista Ordenada (tag <ol>) cria uma sequência numérica:

  1. Primeiro passo
  2. Segundo passo

Em ambas, usamos a tag <li> para representar cada item.

(Voltar ao início)

5. Links

Por fim, aprendi a criar hiperlinks usando a tag <a> (âncora). O atributo href diz para onde o link deve ir.

O atributo target é muito importante: podemos usar _blank para abrir em uma nova aba ou _self para abrir na mesma aba.
